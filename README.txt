Det her er et forsøg på at lære git.

Git er et Version Control System, som bruges i teams.

Hver team har et hoved repository og hver team medlem har deres eget.
De enkelte team medlemmer kan lave et pull fra hoved repositorien.
Her kan de starte deres projekt til de synes det er godt nok til at
at lægges op på hoved repositorien.

Den overordnede af hoved repositorien kan så pulle de andres projekter
og pushe det til hoved repositorien.

Hver medlem gør følgende når de har ændringer.

git pull "Hoved repositorien" (gøres kun når repositorien ændres)

git add .

git commit
